## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The app will open at `http://localhost:3000`

## Design Philosophy

The app emphasizes a warm, tactile aesthetic with amber/orange color schemes and paper-like textures to simulate the experience of writing in a physical journal while leveraging the benefits of digital tools.
