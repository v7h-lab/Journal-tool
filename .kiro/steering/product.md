# Product Overview

Digital Journal Tool is a web-based journaling application that allows users to create and customize digital journal entries with rich media support.

## Core Features

- **Journal Cover Customization**: Users can customize their journal cover with colors, stickers, textures, and titles
- **Page Editing**: Write journal entries with customizable fonts (handwritten styles), font sizes, and rich text
- **Media Support**: Embed images and YouTube videos directly into journal pages
- **Page Customization**: Choose page colors and materials (lined, dotted, grid, or blank)
- **Preview Mode**: View the complete journal in a book-like format

## User Modes

The application has three primary modes:
1. **Edit Mode**: Write and edit journal content with a floating toolbar
2. **Customize Mode**: Personalize journal cover and page appearance
3. **Preview Mode**: View the journal as a finished product

## Design Philosophy

The app emphasizes a warm, tactile aesthetic with amber/orange color schemes and paper-like textures to simulate a physical journal experience in digital form.
